The Core Pocket Media Player 0.72RC1 mod

-- 概要 --
  TCPMP 0.72RC1の改造版です。
  Win32用にI/Fを中心に改造してます。

-- 構成 --
  /src　              tcpmp.src.0.72RC1.tar.bz2から修正したソースファイル
  /tcpmp-mod          modのバイナリファイル
  config-org2mod.vbs  オリジナル版からmodへの設定移行スクリプト
  config-reg2ini.vbs  modの設定をiniファイルに保存するスクリプト
  readme-mod.txt      このファイル

-- インストール --
  (1) tcpmp.win32.0.72RC1.zip をインストール
  (2) language.tgzを削除
  (3) modを上書き
  (4) config-org2mod.vbsを実行し、オリジナル版の設定を移行
      一度行えば、2回目以降の実行は不要
  (5) iniファイルにて設定を保存したい場合、config-reg2ini.vbsを実行して
      出力されたplayer.iniもしくは空のplayer.iniを、
      player.exeと同じ位置に置く。

-- 開発環境 --
  Visual C++ 2005 Express Edition SP1
  Microsoft Platform SDK 2003 SP1
  YASM 0.8.0
